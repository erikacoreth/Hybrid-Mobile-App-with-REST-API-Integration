**React Native | Expo | Google Maps API | OpenWeather API**

This project is a cross-platform mobile application built using React Native with Expo that displays an interactive Google Map and retrieves real-time weather data for a selected location. The app integrates the Google Maps API for map rendering and the OpenWeather REST API to fetch and display live weather information.

This project represents the hybrid mobile implementation of a multi-platform system originally developed as a web app (Weeks 4‚Äì5) and a native iOS app (Week 6).
